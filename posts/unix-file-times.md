### `ctime`

- when the file was last changed.
- To view:
 
        # ls -lc
 
### `atime`

- when the file was last accessed (e.g. `cat`/`open`).
- To view:

        # ls -lu

### `mtime`

- when the contents of the file was last modified.
- To view:

        # ls -l
